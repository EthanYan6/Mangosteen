# 山竹使用手册

本手册分两章：

1. **第一章 刷机教程**：如何用电脑网站给手台刷固件、字库与校准等  
2. **第二章 系统使用手册**：刷好山竹固件后的基本操作、收音机（WFM）、扫描、存台、命名与短信

> 仅适配泉盛 **UV-K1 / UV-K5 / UV-K6 的 V3 版本**。请先确认机身标签有 **V3** 字样。

---

# 第一章 刷机教程

欢迎使用 **山竹（Mangosteen）刷机网站**。本章说明如何在浏览器中完成备份校准、刷固件、刷字库、校验/恢复校准、写频与开机图片。

## 1.1 准备工作

1. 使用 **Chrome** 或 **Edge** 浏览器（需支持 Web Serial）。
2. 准备原厂编程线（USB 转串口），正确连接手台与电脑。
3. 首次刷入第三方固件前，务必在**原厂固件、正常开机使用界面**下先完成一次**备份校准**。

## 1.2 推荐刷机顺序

初次刷入山竹时建议按此顺序：

1. **备份校准**（原厂固件、正常开机）
2. **刷固件**（需进入 BOOT 模式）
3. **刷字库**（正常开机；中文菜单/信道名需要）
4. **校验校准**（可选）
5. **恢复校准**（正常开机，写入此前备份）
6. **写频 / 开机图片**（按需）

## 1.3 备份校准

**状态要求：** 手台正常开机并进入使用界面（能显示信道、菜单），**不要**进入 BOOT 刷机界面。

1. 打开网站，切换到「备份校准」页签。
2. 点击「导出校准数据」，在浏览器弹窗中选择串口并连接。
3. 导出成功后下载 `calibration.dat`，妥善保存。

注意：只需在原厂固件刷机前备份一次。后续请优先使用这份原始备份做恢复。

## 1.4 刷固件

**状态要求：** 进入 **BOOT 模式**。

进入 BOOT：关机后，**按住 PTT 键**，同时转动旋钮开机。

1. 切换到「刷固件」页签。
2. 「远程获取」下载最新 `mangosteen_*.bin`，或「本地选择」已有固件。
3. 确认机型为 V3 后，点击「刷入固件」，选择串口并等待完成。
4. 刷写过程中请勿拔线、关页或断开电源。

## 1.5 刷字库

**状态要求：** 正常开机进入使用界面（**无需 BOOT**）。

1. 切换到「刷字库」页签。
2. 「远程获取」加载本站默认合并字库，或本地选择 `.bin`。
3. 点击「刷入字库」并等待完成。

字库布局：16×16 于 `0xA0000`，8×8 于 `0xE0000`。未刷字库时中文菜单、信道名、短信可能显示异常。

## 1.6 校验校准

1. 正常开机后连接 USB。
2. 打开「校验校准」，可读取设备校准、加载备份并对比。
3. 确认无误后再按需写入对应地址。

山竹校准固定地址为 `0xB000`。

## 1.7 恢复校准

**状态要求：** 正常开机进入使用界面。

1. 切换到「恢复校准」。
2. 选择此前备份的 `.dat`（512 字节，本站导出格式）。
3. 点击「恢复校准」；完成后设备通常会自动重启。

## 1.8 写频

**状态要求：** 正常开机进入使用界面。

1. 切换到「写频」。
2. 可从设备读取信道，或在表格中编辑后写入。
3. 支持导入 / 导出 Excel；信道名支持英文与中文（GB2312，最长约 10 字节）。
4. 读写结束后断开连接是正常现象。

写频仅适配 **Mangosteen** 固件。调制含 FM / AM / USB / WFM。

## 1.9 开机图片

**状态要求：** 正常开机（无需 BOOT）。按「开机图片」页签提示制作并写入，重启后查看。

## 1.10 刷机常见问题

### 浏览器没有串口 / 找不到设备

请改用 Chrome 或 Edge，确认编程线驱动正常。

### 刷固件连接失败

确认已进入 BOOT（按住 PTT 开机）；换线、换 USB 口后重试。

### 刷完花屏 / 雪花屏

多为校准异常。用刷机前保存的原始 `calibration.dat`，在正常开机状态下恢复校准。

### 中文显示异常或缺字

请确认已刷入本站字库。

### 远程获取失败

可改用「本地选择」，手动下载 Releases 中的固件后再刷。

## 1.11 安全提示

- 确认机型为 **V3**。
- 刷写时保持供电与 USB 稳定。
- 重要数据（校准、信道）请提前备份。
- 本站通过浏览器 Web Serial 与设备通信，不会上传你的校准或信道文件。

# 第二章 系统使用手册

本章介绍山竹固件开机后的日常使用。键盘说明：机身 **`#` 键即 F 键**（短按进入 F 组合键，长按锁屏）；**`*` 键**常用于扫描与输入法切换。

## 2.1 基本操作

### 开关机与音量

- **开关机 / 音量**：用机身音量旋钮。关机后按住 **PTT** 再拧旋钮开机，会进入 BOOT（仅刷机时使用）。
- 菜单中的「设置音量」调节扬声器增益，日常听感仍以旋钮为主。

### 打开菜单

1. 短按 **MENU** 进入菜单（中文界面标题为「菜单」）。
2. **↑ / ↓** 浏览项目，也可直接按数字跳到对应编号。
3. 再按 **MENU** 进入子项；**EXIT** 返回或退出。
4. 危险操作会出现「确认?」，再按 **MENU** 执行。

菜单语言可在 **Lang（语言）** 中切换中文 / 英文。

### 频率模式（VFO）与信道模式（MR）

| 操作 | 按键 |
|------|------|
| 切换 VFO ↔ 信道 | **F + 3**（先短按 `#`，再按 `3`） |
| 信道模式下复制到 VFO | **F + 1** |
| VFO 下切换波段 | **F + 1** |

- **VFO**：↑/↓ 按步进改频率；可直接输入数字改频。
- **信道（MR）**：↑/↓ 换信道；可输入信道号跳转。

### A / B 双频与接收模式

| 操作 | 按键 |
|------|------|
| 切换 A ↔ B | **F + 2** |

若菜单 **接收模式** 设为「仅主信道」（MAIN ONLY），再按 **F+2** 会拒绝切换并蜂鸣——此时没有第二路可切换。

接收模式（菜单 **接收模式 / RxMode**）常见选项：

| 模式 | 含义 |
|------|------|
| 仅主信道 | 只使用主频；不能 A/B 切换 |
| 双频守候 | 两路守候，有信号时可把主频切到有信号的一面 |
| 跨段 | 主频发、副频收等跨段用法 |
| 主发双收 | 始终主频发射，两路都可收 |

### 发射、功率与调制

- **发射**：按住侧键 **PTT**。
- **切换功率**：可试 **F + 6**，或侧键功能「POWER」，以及菜单「功率」。
- **调制模式**（FM / AM / USB / WFM 等）：侧键「MODE」或菜单「Mode」。听广播电台见 **2.2 收音机如何使用**。

### 常用 F 组合键（主界面）

| 组合 | 功能 |
|------|------|
| F + 2 | 切换 A/B |
| F + 3 | VFO / 信道 |
| F + 4 | 频率/亚音扫描界面 |
| F + 5 | 扫描列表 / 扫描范围相关 |
| F + 6 | 发射功率 |
| F + 7 | HEARD / 距离检测（信息相关） |
| F + MENU | 打开短信（信息） |
| 长按 * | 开始 / 停止信道或频率扫描 |
| F + * | CTCSS/DCS 亚音搜索 |

侧键 **F1 / F2 / 长按 M** 等可在菜单里自定义为扫描、监听、信息、HEARD、仅主信道等功能。

## 2.2 收音机如何使用

山竹用主界面的 **WFM** 调制收听调频广播（约 **64.0–108.0 MHz**），由机内 BK1080 芯片解调。没有单独的「收音机」全屏界面。

### 进入收听

任选其一：

1. **自动**：在 VFO 下把频率调到或输入 **64.0–108.0 MHz**，调制会自动切为 **WFM**。
2. **手动**：侧键「MODE」或菜单「Mode」循环到 **WFM**（也可在该频段外手动选 WFM）。

用音量旋钮调节听感；**↑ / ↓** 或数字键改频。

### 存台与扫描

- 调好频率后，可按 **2.4 如何保存信道** 存入记忆信道（调制会记为 WFM）。
- 在 WFM、VFO 下 **长按 `*`** 可搜台；搜到电台后会停下。停止扫描仍用短按 **EXIT** / **MENU**。

### 注意

- **64–108 MHz 禁止发射**（任意调制按 PTT 都会被拒绝）；WFM 本身也只收不发。
- 任一 VFO 处于 WFM 时，双频守候会暂停。
- 在频段内可手动改成 FM / AM / USB 等，继续在段内调频时不会强行改回 WFM。
- 调出 64–108 MHz 且当前仍为 WFM 时，会自动回到 **FM**。

## 2.3 如何扫描

### 开始 / 停止扫描

1. 在主界面 **长按 `*`**，或按下已绑定为「SCAN」的侧键，开始扫描。
2. 扫描中短按 **EXIT** 或 **MENU** 可停止。
3. 扫描中 **↑ / ↓** 可改变扫描方向。
4. 在信道模式下，扫描中再次按「扫描」可切换扫描列表。

### 扫描类型说明

- **信道扫描（MR）**：按当前扫描列表依次扫已存信道。
- **频率扫描（VFO）**：按当前步进扫频率。
- **扫描范围**：在 VFO 下用 **F+5** 设定范围（与另一路 VFO 频率相关），再长按 `*` 在该范围内扫描；再按 F+5 可清除范围。
- **亚音搜索**：**F + `*`** 搜索 CTCSS/DCS。
- **F+4**：进入频率/亚音扫描专用界面。

### 跳过 / 排除

- 扫描停在某个信道时，**长按 MENU** 可将该信道排除并继续扫。
- **F+5** 可调整该信道是否参与扫描列表。

### 相关菜单

| 菜单（中文） | 作用 |
|--------------|------|
| 扫描列表 | 选择默认列表 / 全部 |
| 优先扫描、优先信道 | 优先信道设置 |
| 扫描恢复 | 遇信号后的恢复方式（停止 / 载波 / 超时等） |
| 信道列表 | 信道是否加入某扫描列表 |

## 2.4 如何保存信道

把当前频率与参数存进记忆信道：

1. 在主界面调好频率、亚音、功率、调制等。
2. 短按 **MENU** → 找到 **存信道（ChSave）** → 再按 **MENU**。
3. 用 **↑/↓** 或输入 4 位信道号，选择目标信道。
4. 按 **MENU** → 出现「确认?」后再按 **MENU** 完成保存。

删除信道：菜单 **删信道（ChDele）**，同样需确认。

也可用网站「写频」页批量导入信道。

## 2.5 命名信道与输入法

### 进入命名

1. **MENU** → **命名信道（ChName）**。
2. 选择要命名的信道（须为有效信道）→ **MENU** 进入编辑。
3. 编辑完成后，短按 **MENU** 移动光标；到末尾再确认「确认?」保存。
4. 信道名最长约 **10 字节**（英文按字符计；中文按 GB2312，约 5 个汉字）。

> 显示中文需要已刷字库。

### 输入模式（命名信道）

在命名界面短按 **`#`（F）** 循环切换：

| 模式角标 | 说明 |
|----------|------|
| `abc` | 小写字母多击输入（2=abc …） |
| `ABC` | 大写字母多击输入 |
| `123` | 纯数字 |
| `pinyin` | T9 拼音 → 选汉字 |

其他说明：

- 非拼音模式下，长按 `#` 可插入 `#` 字符。
- 拼音模式下，`*` 为退格；其他模式 `*` 可输入 `-` 等。
- 长按数字键可强制输入该数字。

### 拼音选字要点

1. 短按 `#` 切到 `pinyin`。
2. 用 **2–9** 按拼音字母对应数字输入（见下表）；例如「ni」按 `64`，「hao」按 `426`。
3. **1** 进入音节/候选确认流程；**↑/↓** 翻页或切换候选。
4. 按候选编号选字；再按 **MENU** 确认（视当前是否在组词中而定）。
5. 退格用 **`*`**；**0** 输入空格。

#### 拼音按键字母对照表

拼音模式按标准手机九键（T9）对应字母：

| 按键 | 对应字母 |
|------|----------|
| **1** | 确认 / 选字（进入候选后按编号选字） |
| **2** | A B C |
| **3** | D E F |
| **4** | G H I |
| **5** | J K L |
| **6** | M N O |
| **7** | P Q R S |
| **8** | T U V |
| **9** | W X Y Z |
| **0** | 空格 |
| **\*** | 退格 |
| **↑ / ↓** | 翻音节 / 翻候选页 |

示例：输入「山竹」→ `shan` = `7426`，`zhu` = `948`，再按提示选字。

呼号、本机信息名称、DTMF 等编辑框一般**不支持拼音**，仅英文/数字类字符。

## 2.6 短信如何使用

山竹支持电台短信（界面标题「信息」），可与兼容的 GOGUFW 等信息协议互通。后台可收信；未读时状态栏可能显示信封图标。

### 打开与退出

| 操作 | 按键 |
|------|------|
| 打开信息 | **F + MENU**（先 `#` 再 MENU），或侧键「MESSENGER」 |
| 打开 HEARD / 距离 | **F + 7** 或侧键「HEARD」 |
| 退回主界面 | 在信息首页按 **EXIT** |

### 主菜单项

用 ↑/↓ 选择，**MENU** 进入：

| 中文 | 英文 | 作用 |
|------|------|------|
| 收件箱 | INBOX | 查看收到的短信 |
| 编辑信息 | COMPOSE | 写新短信 |
| 发件箱 | SENT | 已发送记录 |
| 草稿箱 | DRAFTS | 草稿 |

### 编写与发送

1. 进入「编辑信息」。
2. 输入正文（最长约 **36** 字节）。
3. 按 **MENU** 发送（空内容会按协议发送空消息标识）。
4. 按 **EXIT** 取消编辑并返回，不发送。

阅读收件后按 **MENU** 可回复（正文前带 `RE: `）。发件箱中按 **MENU** 可重发。列表或阅读界面按 **`#`（F）** 可删除（界面有删除提示）。

注意：

- **收件箱 / 发件箱在内存中**，关机后可能清空。
- **草稿、本机呼号**会写入 Flash，可保留。
- 请在菜单设置 **信息本机名称（MsgCsg）**，便于对方识别你。

### 短信输入法

编辑短信时，短按 **`*`** 循环切换：

| 角标 | 模式 |
|------|------|
| `B` | 大写字母多击 |
| `b` | 小写字母多击 |
| `2` | 数字 |
| `pinyin` | 拼音选字（与命名信道同一套引擎，按键字母对照见 **2.5 拼音按键字母对照表**） |

- 退格：按 **`#`（F）**。
- 字母模式下长按数字，可插入数字而不离开字母模式。
- `0` 多为空格；`1` 在字母模式多为标点。

### 信息相关菜单（主菜单）

| 中文 | 作用 |
|------|------|
| 信息接收 | 开关接收短信 |
| 信息本机名称 | 编辑本机呼号/名称（约 6 字符） |
| 信息应答 | 是否自动应答 |
| 信息接收音 | 收到短信是否响铃 |
| 信息接收灯光 | 收到短信是否亮灯 |
| 信息范围回应 | 距离/范围检测相关回应 |

### HEARD / 距离检测

**F+7** 进入后，用 **MENU** 开始/停止测距或刷新；↑/↓ 浏览听到的台站（呼号、信号、时间等）；**EXIT** 退出。

## 2.7 使用注意

1. **仅主信道**模式下无法 A/B 切换。
2. 中文显示依赖字库，请按第一章完成刷字库。
3. 扫描时双守候界面可能暂时按单频扫描逻辑工作，停止扫描后恢复。
4. 短信收件箱关机可能丢失，重要内容请及时记下；先设好本机信息名称再发信。
5. **64–108 MHz** 及 **WFM** 仅接收，不能发射；详见 **2.2**。
